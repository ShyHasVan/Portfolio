Key Activities:

  • Performed extensive data cleaning and normalization to prepare the dataset for analysis.
  • Conducted Exploratory Data Analysis (EDA) to uncover initial patterns and correlations.
  • Developed and evaluated several predictive models using Scikit-learn, including Logistic Regression and Random Forest.
  • Provided actionable insights that could inform customer retention strategies and reduce churn.
